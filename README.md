## Installation

All installation steps are performed in the root directory of the project:

1. Install NPM:
```
   make install
```
2. Publish package locally:
```
  make publish
```
3. Install the package globally:
```
  sudo npm link
```
4. Start the game:
```
  brain-{name}
```
